問題3-2: find/any?/all?の活用
課題内容
在庫管理システムで、商品の在庫状況をチェックするメソッドを実装してください。
要件
以下の4つのメソッドを実装してください：

find_out_of_stock(products)

商品配列から在庫切れ（:stock が 0）の最初の商品を返す
在庫切れ商品がない場合は nil を返す
find メソッドを使って実装


has_low_stock?(products, threshold)

商品配列の中に、在庫数が指定された閾値以下の商品があるか判定
ある場合は true、ない場合は false を返す
any? メソッドを使って実装


all_in_stock?(products)

全ての商品が在庫あり（:stock が 1以上）か判定
全て在庫ありなら true、1つでも在庫切れなら false を返す
all? メソッドを使って実装


none_expensive?(products, price_limit)

全ての商品が指定価格未満か判定
全て価格限度未満なら true、1つでも以上なら false を返す
none? メソッドを使って実装



テストケース
rubyproducts = [
  { name: "商品A", price: 1000, stock: 5 },
  { name: "商品B", price: 2000, stock: 0 },
  { name: "商品C", price: 1500, stock: 3 },
  { name: "商品D", price: 3000, stock: 1 }
]

# find_out_of_stock のテスト
find_out_of_stock(products)
# => { name: "商品B", price: 2000, stock: 0 }

products_with_stock = [
  { name: "商品A", price: 1000, stock: 5 },
  { name: "商品C", price: 1500, stock: 3 }
]

find_out_of_stock(products_with_stock)
# => nil

# has_low_stock? のテスト
has_low_stock?(products, 2)
# => true  （商品Dの在庫が1）

has_low_stock?(products, 0)
# => true  （商品Bの在庫が0）

has_low_stock?(products_with_stock, 2)
# => false  （全て在庫3以上）

# all_in_stock? のテスト
all_in_stock?(products)
# => false  （商品Bが在庫切れ）

all_in_stock?(products_with_stock)
# => true  （全て在庫あり）

# none_expensive? のテスト
none_expensive?(products, 5000)
# => true  （全て5000円未満）

none_expensive?(products, 2500)
# => false  （商品Dが3000円）

none_expensive?(products_with_stock, 1200)
# => false  （商品Cが1500円）
ヒント

find: 最初にマッチした要素を返す（前回の復習）
any?: 1つでも条件に合えば true
all?: 全て条件に合えば true
none?: 全て条件に合わなければ true（all? の逆）

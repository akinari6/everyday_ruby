問題4-1: yieldを使ったイテレータ
課題内容
独自のイテレータメソッドを作成し、yield を使ってブロックを実行する仕組みを理解しましょう。
要件
以下の3つのメソッドを実装してください：

repeat(n)

与えられたブロックをn回実行する
現在の回数（1から始まる）をブロックに渡す
yield を使って実装


each_with_index_from_one(array)

配列の各要素とインデックス（1から始まる）をブロックに渡す
yield を使って実装
Rubyの each_with_index は0から始まるが、これは1から始まる


measure_time

ブロックの実行時間を計測して表示する
ブロック実行前に "開始" と表示
ブロック実行後に "完了: X秒" と表示（小数点以下2桁まで）
yield を使って実装
時間計測には Time.now を使う



テストケース
ruby# repeat のテスト
repeat(3) do |i|
  puts "#{i}回目の実行"
end
# 出力:
# 1回目の実行
# 2回目の実行
# 3回目の実行

# each_with_index_from_one のテスト
each_with_index_from_one(["りんご", "バナナ", "みかん"]) do |fruit, index|
  puts "#{index}番目: #{fruit}"
end
# 出力:
# 1番目: りんご
# 2番目: バナナ
# 3番目: みかん

# measure_time のテスト
measure_time do
  sleep(0.5)
  puts "処理中..."
end
# 出力:
# 開始
# 処理中...
# 完了: 0.50秒
ヒント

yield: メソッド内で yield を呼ぶと、渡されたブロックが実行される
yield に引数を渡す: yield(value) でブロックに値を渡せる
Time.now: 現在時刻を取得
時間の差: Time.now - start_time で経過秒数
フォーマット: format("%.2f", number) で小数点以下2桁

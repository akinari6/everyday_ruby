問題5-1: 基本的なクラス設計
課題内容
書籍を管理するための Book クラスを実装してください。Rubyのクラスの基本構文を理解することが目標です。
要件
以下の仕様で Book クラスを実装してください：

初期化メソッド (initialize)

引数: title（タイトル）、author（著者）、price（価格）
これらをインスタンス変数に保存


インスタンス変数の読み取りメソッド

title、author、price を読み取れるようにする
ゲッターメソッドを手動で定義（attr_* は使わない）


インスタンスメソッド

info: 書籍情報を文字列で返す

形式: "『タイトル』著者名 - 価格円"


discounted_price(rate): 割引後の価格を返す

rate は割引率（0.0〜1.0、例: 0.1 = 10%引き）
整数で返す（小数点以下切り捨て）


expensive?: 価格が3000円以上なら true、未満なら false



テストケース
ruby# Book クラスのインスタンス作成
book1 = Book.new("Ruby入門", "山田太郎", 2800)
book2 = Book.new("Rails実践ガイド", "佐藤花子", 3500)

# ゲッターメソッドのテスト
p book1.title
# => "Ruby入門"

p book1.author
# => "山田太郎"

p book1.price
# => 2800

# info メソッドのテスト
p book1.info
# => "『Ruby入門』山田太郎 - 2800円"

p book2.info
# => "『Rails実践ガイド』佐藤花子 - 3500円"

# discounted_price メソッドのテスト
p book1.discounted_price(0.1)
# => 2520  (2800 * 0.9 = 2520)

p book2.discounted_price(0.2)
# => 2800  (3500 * 0.8 = 2800)

# expensive? メソッドのテスト
p book1.expensive?
# => false  (2800 < 3000)

p book2.expensive?
# => true   (3500 >= 3000)
ヒント

クラス定義: class ClassName ... end
初期化: def initialize(args) ... end
インスタンス変数: @variable_name で定義
ゲッターメソッド: インスタンス変数の値を返すメソッド

ruby  def title
    @title
  end

メソッド内でインスタンス変数にアクセス: @title で直接アクセス可能
真偽値メソッド: ? で終わるメソッド名は真偽値を返す慣習

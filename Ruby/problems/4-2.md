問題4-2: ブロック引数の活用
課題内容
ブロックに複数の引数を渡したり、ブロックの戻り値を活用するメソッドを実装しましょう。
要件
以下の3つのメソッドを実装してください：

transform_array(array)

配列の各要素とそのインデックスをブロックに渡す
ブロックの戻り値を新しい配列として返す
map のような動作をするが、インデックスも渡す
yield を使って実装


select_with_index(array)

配列の各要素とそのインデックスをブロックに渡す
ブロックが true を返した要素だけを集めた配列を返す
select のような動作をするが、インデックスも使える
yield を使って実装


build_hash(keys)

キーの配列を受け取る
各キーをブロックに渡し、ブロックの戻り値を値とするハッシュを作成
yield を使って実装



テストケース
ruby# transform_array のテスト
result = transform_array([10, 20, 30]) do |value, index|
  "#{index}: #{value}"
end
p result
# => ["0: 10", "1: 20", "2: 30"]

result = transform_array([1, 2, 3]) do |value, index|
  value * index
end
p result
# => [0, 2, 6]

# select_with_index のテスト
result = select_with_index([10, 20, 30, 40, 50]) do |value, index|
  index.even?
end
p result
# => [10, 30, 50]  （インデックス0, 2, 4）

result = select_with_index([10, 20, 30, 40]) do |value, index|
  value > 20 && index > 1
end
p result
# => [30, 40]

# build_hash のテスト
result = build_hash([:name, :age, :city]) do |key|
  case key
  when :name then "太郎"
  when :age then 25
  when :city then "東京"
  end
end
p result
# => { name: "太郎", age: 25, city: "東京" }

result = build_hash([1, 2, 3]) do |key|
  key * 10
end
p result
# => { 1 => 10, 2 => 20, 3 => 30 }
ヒント

yield の戻り値: result = yield(args) でブロックの戻り値を受け取れる
配列の作成: [] で空配列を作り、<< で要素を追加
ハッシュの作成: {} で空ハッシュを作り、hash[key] = value で追加
each_with_index: 使ってもOK（ただし yield は必須）

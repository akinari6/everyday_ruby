問題2-3: ユーザーデータの検索とフィルタリング
課題内容
ユーザーの配列から、条件に合うユーザーを検索・抽出するメソッドを実装してください。
要件
以下の3つのメソッドを実装してください：

find_user_by_id(users, id)

ユーザー配列から指定されたIDのユーザーを検索
見つかった場合はそのユーザーハッシュを返す
見つからない場合は nil を返す
find メソッドを使って実装


filter_adults(users)

ユーザー配列から20歳以上のユーザーだけを抽出
抽出したユーザーの配列を返す
select メソッドを使って実装


has_admin?(users)

ユーザー配列の中に :role が :admin のユーザーがいるか判定
いる場合は true、いない場合は false を返す
any? メソッドを使って実装



テストケース
rubyusers = [
  { id: 1, name: "田中", age: 25, role: :user },
  { id: 2, name: "佐藤", age: 17, role: :user },
  { id: 3, name: "鈴木", age: 30, role: :admin },
  { id: 4, name: "高橋", age: 22, role: :user }
]

# find_user_by_id のテスト
find_user_by_id(users, 3)
# => { id: 3, name: "鈴木", age: 30, role: :admin }

find_user_by_id(users, 99)
# => nil

# filter_adults のテスト
filter_adults(users)
# => [
#      { id: 1, name: "田中", age: 25, role: :user },
#      { id: 3, name: "鈴木", age: 30, role: :admin },
#      { id: 4, name: "高橋", age: 22, role: :user }
#    ]

# has_admin? のテスト
has_admin?(users)
# => true

users_without_admin = [
  { id: 1, name: "田中", age: 25, role: :user },
  { id: 2, name: "佐藤", age: 17, role: :user }
]

has_admin?(users_without_admin)
# => false
ヒント

find: array.find { |item| condition } で最初にマッチした要素を返す
select: array.select { |item| condition } で条件に合う全要素を返す
any?: array.any? { |item| condition } で条件に合う要素が1つでもあれば true
ハッシュの値比較には == を使います

学習ポイント
この問題を通して以下を学びます：

find と select の違い（1つ vs 複数）
any? による存在チェック
ブロック内での条件判定
配列とハッシュの組み合わせ操作
? で終わるメソッド名（真偽値を返す慣習）

# 問題5-3: 継承とモジュール

## 課題内容

継承とモジュールのMixinを使って、コードの再利用性を高める方法を学びましょう。

### 要件

以下の仕様でクラスとモジュールを実装してください：

1. **`Discountable` モジュール**
   - `apply_discount(rate)` メソッドを提供
   - 自分自身の `price` に割引率を適用した価格を返す（整数）
   - `rate` は割引率（0.0〜1.0）

2. **`Item` クラス（親クラス）**
   - 属性: `name`（読み取り専用）、`price`（読み取り専用）
   - 初期化: キーワード引数で `name` と `price` を受け取る
   - メソッド: `info` - `"商品名: 価格円"` の形式で返す

3. **`Book` クラス（`Item` を継承）**
   - `Discountable` モジュールをMixin
   - 追加属性: `author`（読み取り専用）
   - 初期化: `name`, `price`, `author` を受け取る（`super` を使う）
   - `info` をオーバーライド: `"書籍: タイトル by 著者 - 価格円"`

4. **`Electronics` クラス（`Item` を継承）**
   - `Discountable` モジュールをMixin
   - 追加属性: `warranty_years`（保証年数、読み取り専用）
   - 初期化: `name`, `price`, `warranty_years` を受け取る
   - `info` をオーバーライド: `"電化製品: 商品名 - 価格円（保証: Y年）"`

### テストケース
```ruby
# Book クラスのテスト
book = Book.new(name: "Ruby入門", price: 2800, author: "山田太郎")

p book.name
# => "Ruby入門"

p book.price
# => 2800

p book.author
# => "山田太郎"

p book.info
# => "書籍: Ruby入門 by 山田太郎 - 2800円"

p book.apply_discount(0.1)
# => 2520  (2800 * 0.9)

# Electronics クラスのテスト
laptop = Electronics.new(name: "ノートPC", price: 120000, warranty_years: 3)

p laptop.name
# => "ノートPC"

p laptop.price
# => 120000

p laptop.warranty_years
# => 3

p laptop.info
# => "電化製品: ノートPC - 120000円（保証: 3年）"

p laptop.apply_discount(0.15)
# => 102000  (120000 * 0.85)

# 継承の確認
p book.is_a?(Item)
# => true

p laptop.is_a?(Item)
# => true

p book.is_a?(Book)
# => true
```

### ヒント
- **モジュールの定義**: `module ModuleName ... end`
- **Mixin**: `include ModuleName` でモジュールをクラスに取り込む
- **継承**: `class Child < Parent`
- **super**: 親クラスのメソッドを呼び出す
  ```ruby
  def initialize(name:, price:, author:)
    super(name: name, price: price)  # 親クラスの initialize を呼ぶ
    @author = author
  end
  ```
- **メソッドのオーバーライド**: 親クラスと同名のメソッドを定義すると上書きされる
- **is_a?**: インスタンスが指定したクラス/モジュールのインスタンスか確認

### 学習ポイント
この問題を通して以下を学びます：
- モジュールの定義と使い方
- `include` による Mixin
- クラスの継承（`<`）
- `super` による親クラスのメソッド呼び出し
- メソッドのオーバーライド
- コードの再利用パターン
- 継承 vs Mixin の使い分け

問題1-2: 後置if/unlessの活用
課題内容
ユーザーの年齢と会員ステータスをチェックして、適切なメッセージを返すメソッドを実装してください。
要件

メソッド名は check_access とする
引数は age（整数）と is_member（真偽値）
以下のルールに従ってメッセージを返す：

年齢が18歳未満の場合: "未成年のためアクセスできません"
会員でない場合: "会員登録が必要です"
両方の条件を満たす場合: "アクセス可能です"


後置if/unlessを使って実装する

テストケース
rubycheck_access(25, true)
# => "アクセス可能です"

check_access(16, true)
# => "未成年のためアクセスできません"

check_access(25, false)
# => "会員登録が必要です"

check_access(16, false)
# => "未成年のためアクセスできません"
ヒント

Rubyでは条件式を文の後ろに置く「後置if」「後置unless」が使えます

ruby  # 通常のif
  if condition
    do_something
  end
  
  # 後置if（1行で書ける時に便利）
  do_something if condition

unless は if not の意味です
早期リターン（ガード節）のパターンを使うと読みやすくなります

問題3-1: map/selectの基本
課題内容
商品データの配列を操作して、必要な情報を抽出・変換するメソッドを実装してください。
要件
以下の3つのメソッドを実装してください：

get_product_names(products)

商品配列から商品名だけを抽出した配列を返す
map メソッドを使って実装


get_expensive_products(products, threshold)

商品配列から指定価格以上の商品だけを抽出した配列を返す
select メソッドを使って実装


get_discounted_prices(products, discount_rate)

商品配列から価格だけを取り出し、割引率を適用した価格の配列を返す
割引率は0.0〜1.0の範囲（例: 0.1 = 10%引き）
map メソッドを使って実装
価格は整数に変換（小数点以下切り捨て）



テストケース
rubyproducts = [
  { name: "ノートPC", price: 120000, category: "電化製品" },
  { name: "マウス", price: 2500, category: "電化製品" },
  { name: "キーボード", price: 8000, category: "電化製品" },
  { name: "モニター", price: 35000, category: "電化製品" }
]

# get_product_names のテスト
get_product_names(products)
# => ["ノートPC", "マウス", "キーボード", "モニター"]

# get_expensive_products のテスト
get_expensive_products(products, 10000)
# => [
#      { name: "ノートPC", price: 120000, category: "電化製品" },
#      { name: "モニター", price: 35000, category: "電化製品" }
#    ]

get_expensive_products(products, 50000)
# => [{ name: "ノートPC", price: 120000, category: "電化製品" }]

# get_discounted_prices のテスト
get_discounted_prices(products, 0.1)
# => [108000, 2250, 7200, 31500]

get_discounted_prices(products, 0.2)
# => [96000, 2000, 6400, 28000]
ヒント

map: array.map { |item| item[:key] } で各要素から特定の値を取り出せます
select: array.select { |item| condition } で条件に合う要素を抽出
割引計算: price * (1 - discount_rate) で割引後の価格
整数変換: to_i メソッドで整数に変換

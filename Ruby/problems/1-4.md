問題2-1: 配列生成の多様な方法
課題内容
Rubyには配列を生成する様々な方法があります。以下の仕様に従って、それぞれ異なる方法で配列を生成するメソッドを実装してください。
要件
以下の4つのメソッドを実装してください：

create_number_array(n)

1からnまでの整数の配列を返す
範囲オブジェクト（Range）を使って実装


create_repeat_array(value, count)

指定した値をcount回繰り返す配列を返す
Array.new を使って実装


create_word_array

["apple", "banana", "cherry", "date"] という配列を返す
%w リテラルを使って実装


create_symbol_array

[:red, :green, :blue, :yellow] という配列を返す
%i リテラルを使って実装



テストケース
rubycreate_number_array(5)
# => [1, 2, 3, 4, 5]

create_number_array(3)
# => [1, 2, 3]

create_repeat_array("hello", 3)
# => ["hello", "hello", "hello"]

create_repeat_array(0, 4)
# => [0, 0, 0, 0]

create_word_array
# => ["apple", "banana", "cherry", "date"]

create_symbol_array
# => [:red, :green, :blue, :yellow]
ヒント

範囲オブジェクト: (1..5) や (1...5) を配列に変換できます
Array.new: Array.new(size, value) や Array.new(size) { block } の形式があります
%w リテラル: %w[word1 word2] でスペース区切りの文字列配列を作成
%i リテラル: %i[symbol1 symbol2] でスペース区切りのシンボル配列を作成

Array.new の様々な使い方

配列生成の多様なアプローチ

問題1-3: 文字列操作とシンボル
課題内容
ユーザー情報のハッシュを受け取り、フォーマットされた自己紹介文を返すメソッドを実装してください。
要件

メソッド名は format_profile とする
引数はハッシュ（キーはシンボル）

:name - 名前（文字列）
:age - 年齢（整数）
:occupation - 職業（文字列、省略可能）


戻り値の形式:

職業がある場合: "こんにちは、[名前]です。[年齢]歳で、[職業]をしています。"
職業がない場合: "こんにちは、[名前]です。[年齢]歳です。"


文字列補間を使って実装する

テストケース
rubyformat_profile(name: "田中太郎", age: 25, occupation: "エンジニア")
# => "こんにちは、田中太郎です。25歳で、エンジニアをしています。"

format_profile(name: "佐藤花子", age: 30)
# => "こんにちは、佐藤花子です。30歳です。"

format_profile(name: "鈴木一郎", age: 22, occupation: "学生")
# => "こんにちは、鈴木一郎です。22歳で、学生をしています。"
ヒント

文字列補間: "文字列 #{変数} 文字列" の形式
ハッシュの値取得: hash[:key] または hash.fetch(:key)
職業の有無で条件分岐が必要です
nil チェックには様々な方法があります（nil?, 三項演算子, ifなど）

学習ポイント
この問題を通して以下を学びます：

文字列補間（#{}）の使い方
シンボルとハッシュのキーの関係
ハッシュからの値取得方法
nil のチェック方法
文字列の連結方法の違い（+ vs 補間）

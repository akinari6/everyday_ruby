# 問題3-3: 売上データの集計

## 課題内容

売上データの配列から、様々な集計値を計算するメソッドを実装してください。

### 要件

以下の4つのメソッドを実装してください：

1. **`total_sales(sales)`**
   - 売上データの合計金額を計算
   - `sum` メソッドを使って実装

2. **`average_sales(sales)`**
   - 売上データの平均金額を計算（小数点以下も含む）
   - 売上データが空の場合は `0.0` を返す
   - 配列のメソッドを組み合わせて実装

3. **`max_sale(sales)`**
   - 最も高い売上金額を返す
   - `max` メソッドを使って実装

4. **`min_sale(sales)`**
   - 最も低い売上金額を返す
   - `min` メソッドを使って実装

### テストケース
```ruby
sales = [
  { product: "商品A", amount: 15000 },
  { product: "商品B", amount: 8000 },
  { product: "商品C", amount: 25000 },
  { product: "商品D", amount: 12000 },
  { product: "商品E", amount: 20000 }
]

# total_sales のテスト
total_sales(sales)
# => 80000

# average_sales のテスト
average_sales(sales)
# => 16000.0

average_sales([])
# => 0.0

# max_sale のテスト
max_sale(sales)
# => 25000

# min_sale のテスト
min_sale(sales)
# => 8000
```

### ヒント
- **sum**: `array.sum` で配列の合計を計算。ブロックも渡せます
  - 例: `array.sum { |item| item[:amount] }`
- **size/length**: 配列の要素数を取得
- **max/min**: `array.max` や `array.min` で最大値・最小値。ブロックも渡せます
- 平均 = 合計 ÷ 要素数
- 空配列のチェックには `empty?` が使えます

### 学習ポイント
この問題を通して以下を学びます：
- `sum` メソッドの使い方
- `max` / `min` メソッドの使い方
- ブロック引数を受け取る集計メソッド
- 平均値の計算方法
- エッジケース（空配列）の処理
- 浮動小数点数の扱い

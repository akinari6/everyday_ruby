# 問題1-1: 配列から条件抽出と変換（修正版）

## 課題内容

整数の配列を受け取り、**偶数だけを抽出して2乗した値の配列**を返すメソッドを実装してください。

### 要件
1. メソッド名は `square_evens` とする
2. 引数は整数の配列
3. 戻り値は偶数を2乗した値の配列
4. 空配列が渡された場合は空配列を返す
5. **Rubyらしい書き方で実装する**

### テストケース
```ruby
# 実行例
square_evens([1, 2, 3, 4, 5, 6])
# => [4, 16, 36]

square_evens([1, 3, 5, 7])
# => []

square_evens([])
# => []

square_evens([2, 4, 6, 8])
# => [4, 16, 36, 64]
```

### ヒント
- Rubyではループを明示的に書く必要はありません
- Enumerableモジュールのメソッドを活用しましょう
- メソッドチェーンを使うとスマートに書けます

### 学習ポイント
この問題を通して以下を学びます：
- Ruby的な配列操作の考え方
- `select`（または`filter`）メソッドの使い方
- `map`メソッドの使い方
- メソッドチェーンの基本
- ブロックの基本的な書き方（`{ }` と `do...end` の使い分け）
- 偶数判定（`even?`メソッド）
